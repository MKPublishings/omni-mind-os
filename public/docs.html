Overview

Omni Mind/OS is a multi‑mode intelligence shell designed to unify reasoning, memory, and creative workflows into a single cinematic interface.

Architecture

The system runs on Cloudflare Pages + Workers, with a streaming API endpoint at /api/omni. The frontend is modular and theme‑driven.

Modes

Each mode represents a cognitive lens: Architect, Lore, Visual, Analyst.

Memory

Omni stores persistent facts, preferences, and codex entries to maintain continuity.

API

The API accepts JSON POST requests with { message, mode, model }.
