Development Workflow

Local Development

# Install dependencies
npm install

# Start local dev server with hot reload
npx wrangler dev

# Run TypeScript compiler
npm run build

Deployment

# Deploy to production
npx wrangler deploy

# Deploy with environment variables
npx wrangler deploy --env production

# View live logs
npx wrangler tail

Environment Configuration

Set model aliases and API keys in wrangler.toml:

[vars]
MODEL_OMNI = "@cf/meta/llama-3.3-70b-instruct-fp8-fast"
MODEL_GPT4O = "@cf/openai/gpt-4o"
MODEL_GPT4O_MINI = "@cf/openai/gpt-4o-mini"
MODEL_DEEPSEEK = "@cf/deepseek-ai/deepseek-r1-distill-llama-70b"

[[kv_namespaces]]
binding = "MEMORY"
id = "your-memory-namespace-id"

[[kv_namespaces]]
binding = "MIND"
id = "your-mind-namespace-id"
