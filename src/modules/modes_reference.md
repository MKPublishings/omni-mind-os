# Modes Reference

## Architect
System structure, implementation sequencing, and technical planning.

## Reasoning
Whole -> Parts -> Synthesis scaffold with final-answer output style.

## Coding
Logic-first explanations, fenced code blocks, and self-review checks.

## Creative
Narrative and concept generation with coherent style control.

## System Knowledge (OS)
Behavior anchored to Omni identity and internal rules/modules.

Mode selection can be explicit from the UI or inferred by the router when automatic routing is enabled.
