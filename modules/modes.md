# Modes

- Architect: system design, planning, architecture.
- Analyst: decomposition, evidence, comparisons.
- Visual: visual descriptions and creative scenes.
- Lore: narrative, worldbuilding, story framing.
- Reasoning: hidden chain-of-thought with final concise answer.
- Coding: logic explanation + fenced code + self-review.
- Knowledge: retrieval-augmented factual answers.
- System Knowledge: uses internal module references.
