# Omni Living Codex (v1)

The codex is a living knowledge graph.

## Chambers
- `laws/`
- `systems/`
- `mythic/`
- `visual-dialect/`
- `equations/`
- `glossary/`

## Runtime
- `codex/index.json` is generated by script.
- `codex/index.schema.json` defines the brain shape.
- `npm run codex:reindex` rebuilds registry + cross-links.
- `npm run codex:register -- <path-to-artifact>` registers a new artifact and refreshes graph.
